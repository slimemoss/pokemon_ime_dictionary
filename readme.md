# ポケモン日本語辞書

![GitHub Release Date](https://img.shields.io/github/release-date/slimemoss/pokemon_ime_dictionary)

このリポジトリは、ポケモンの名前やデータを日本語で扱うための辞書データを提供します。
元データは [Purukitto/pokemon-data.json](https://github.com/Purukitto/pokemon-data.json) から取得しています。

## データ内容
- ポケモンの名前
- 技名
